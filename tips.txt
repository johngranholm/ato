Setup, run, commit
1. Fix git on PATH (the real fix for item 4). Easiest permanent fix: re-run the Git for Windows installer → on the PATH screen choose "Git from the command line and also from 3rd-party software." The new config.find_git() will also locate it automatically even if it's not on PATH, so the supervisor message should disappear either way. Verify in your Python terminal:

cmd
git --version
2. Run:

cmd
python -m ato
3. Commit:

cmd
git add ato/config.py ato/kernel/safety.py ato/kernel/bootstrap.py ato/supervisor.py ato/core/selfedit.py ato/core/agent_loop.py ato/core/archive.py ato/core/gaze.py ato/runtime.py ato/persona/ui.html
git commit -m "Remove restrictions, script archive, gaze bridge, git locator, no step limit, GPT-5.5"
git push
git branch -f lastgood HEAD && git push origin lastgood
How to actually use the gaze feature
Tell A.T.O. something like:

"Edit my intruderAlert app so that while it's running, every frame it writes the detected person's normalized screen position to C:\ato\state\gaze.json as {"x":..,"y":..,"label":"intruder"}. Then run it, and your avatar should turn to look wherever it detects motion."

A.T.O. now has the convention baked into its prompt, the watcher is native, and the avatar's lookAt is wired. When intruderAlert dumps coordinates, the A.T.O. avatar you talk to will physically turn its head toward the detected location.
